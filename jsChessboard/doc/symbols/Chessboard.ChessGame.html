set(code, sq)
Sets a piece on a square of the chessboard.

The piece code is a String of the form XY where:

  • X is the piece color:
    • 'w' for white
    • 'b' for black
  • Y is the piece type
    • 'k' for King
    • 'q' for Queen
    • 'r' for Rook
    • 'b' for Bishop
    • 'n' for Knight
    • 'p' for Pawn

The square code uses the traditional chess square notation: 'a1', 'f6' or 'h8' are valid squares.

Examples:

  • set('wq', 'd1')
    puts a white queen on square 'd1'
  • set('bn', 'c6')
    puts a black knight on square 'c6'
Parameters:
{String} code
the code of the piece to set.
{String} sq
the code
